Cat Laser Pointer

A fun and interactive project to keep your cat entertained using an Arduino and servo motors.

The project came to be when my cat, Ruda, decided that 3 a.m. was the perfect time to practice parkour on the furniture. Desperate for a solution (and some sleep), I realized I needed a way to keep them entertained during the day. Enter the Cat Laser Pointer—a project born out of equal parts necessity and curiosity. Now, Ruda spends their time chasing a laser instead of knocking over my coffee mug. Mission accomplished!

How It Works

The project uses two servo motors to control the movement of the laser pointer. The servos are programmed to move the laser in random directions, creating unpredictable patterns that cats love to chase. The Arduino generates the random movements and controls the servos accordingly.

Components

  • Arduino Uno
  • 2 Servo Motors
  • Laser Pointer
  • Breadboard
  • Jumper Wires
  • Power Supply

3D Design

The mechanical parts for this project were designed in SketchUp to ensure precise fitting and functionality. The design includes a servo base, a servo mount, and a laser holder, all of which were 3D-printed for durability and accuracy. These custom parts make it easy to assemble the laser pointer and ensure smooth movement of the servos.

Steps to Build

  1. Connect the two servo motors to the Arduino using the breadboard and jumper wires.
  2. Attach the laser pointer to the servo motors so it moves with the servos.
  3. Upload the Arduino code to generate random movements for the servos.
  4. Power the Arduino and watch the laser pointer move in random patterns.

Code

Here’s the Arduino code for the Cat Laser Pointer:


#include 

Servo servoX;
Servo servoY;

void setup() {
    servoX.attach(9);
    servoY.attach(10);
    randomSeed(analogRead(0));
}

void loop() {
    int posX = random(0, 180);
    int posY = random(0, 180);

    servoX.write(posX);
    servoY.write(posY);

    delay(500);
}

Conclusion

The Cat Laser Pointer project showcases the creative potential of combining technology and everyday challenges. It’s an engaging way to explore Arduino programming and servo motor control while providing endless entertainment for Ruda.
