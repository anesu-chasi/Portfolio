Johnson's Counter with Auto-Correction

The Johnson's Counter is a sequential circuit that leverages the toggling behavior of JK flip-flops to generate a predictable sequence of states. This design incorporates auto-correction logic to reset the sequence if an invalid state is detected, ensuring robust operation.

Key Features:

  • Sequential Logic: Four JK flip-flops are connected in series, with each stage toggling based on the state of the preceding stage.
  • Auto-Correction: Gating logic ensures that any invalid state immediately resets the sequence, maintaining system integrity.
  • Strict Activation Order: The circuit enforces a strict order of activation, requiring the previous flip-flop's output to be high before the next stage toggles.
  • Precision and Security: Only the correct sequence of inputs propagates through all stages, producing a final active output.

This project demonstrates the practical application of JK flip-flops in designing secure and precise sequence generators.

Moore-Type Checker for Divisibility by 2 or 3

This project showcases a Moore-type finite state machine (FSM) designed to check the divisibility of binary numbers by 2 or 3. The circuit employs sequential logic to transition between states based on the input bits, demonstrating the principles of state machine design.

Key Features:

  • State Machine Design: A Moore FSM with well-defined states for tracking divisibility conditions.
  • Sequential Logic: Uses flip-flops and combinational logic to transition between states based on input bits.
  • Real-Time Evaluation: Processes binary inputs in real-time to determine divisibility.
  • Educational Value: Demonstrates the practical application of FSMs in digital circuit design.

This project highlights the versatility of state machines in solving complex digital logic problems.
