Exchange Rate Viewer (NBP API)

A robust Java application for real-time currency exchange rate retrieval and visualization

The Exchange Rate Viewer is a feature-rich Java application that leverages the Narodowy Bank Polski (NBP) API to fetch and display real-time currency exchange rates. This project demonstrates advanced integration of RESTful APIs, JSON parsing, and GUI development, showcasing a seamless blend of backend and frontend technologies.

Technical Overview

The application is built using Java's core libraries for networking and JSON handling. It sends HTTP GET requests to the NBP API, retrieves JSON responses, and parses them to extract relevant data fields such as currency codes, exchange rates, and dates. The parsed data is dynamically rendered in a user-friendly graphical interface, designed using JavaFX.

Key Features

  • Real-Time Data Retrieval: Fetches up-to-date exchange rates directly from the NBP API.
  • Dynamic Date Range Selection: Allows users to specify custom date ranges for historical data analysis.
  • Interactive User Interface: Built with JavaFX, featuring a responsive ListView for displaying exchange rates.
  • Data Export Functionality: Enables users to save exchange rate data in CSV format for offline analysis.
  • Error Handling: Implements robust exception handling for network errors and invalid API responses.

Implementation Details

The application architecture follows a modular design pattern, separating concerns into distinct layers:

  • API Layer: Handles HTTP requests and responses using Java's HttpURLConnection class.
  • Data Parsing Layer: Utilizes the org.json library to parse JSON responses and map them to Java objects.
  • UI Layer: Built with JavaFX, featuring a clean and intuitive interface for displaying data.

Additionally, the application employs multithreading to ensure a responsive user experience by offloading network operations to background threads.

Use Case Scenarios

This application is ideal for financial analysts, currency traders, or anyone needing quick access to exchange rate data. Its ability to fetch historical data and export it for further analysis makes it a versatile tool for both personal and professional use.
