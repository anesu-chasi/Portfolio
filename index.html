Microprocessor-Based Stopwatch

An 8051 assembly-based stopwatch with start/pause, reset, and custom timer settings. It uses Timer0 interrupts to count time accurately and converts binary values to BCD for display. Debounced button inputs and audible alerts ensure reliable operation and user feedback.

Inverse Polish Notation Calculator

An IPN calculator that converts infix expressions to postfix (Reverse Polish Notation) using Verilog. It employs a state machine and an external stack module to manage operator precedence and token processing. This design showcases efficient hardware-based parsing for arithmetic expressions.
