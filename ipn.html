Inverse Polish Notation (IPN) Calculator

A Verilog-based calculator for evaluating arithmetic expressions in reverse Polish notation using a finite state machine.

The IPN Calculator is a cutting-edge digital design project that demonstrates the power of hardware description languages (HDLs) in solving computational problems. By leveraging Verilog, this project implements a robust finite state machine (FSM) to evaluate arithmetic expressions in reverse Polish notation (RPN). The design employs a stack-oriented approach for sequential processing of operands and operators, ensuring efficient state-based input capture and error detection.

Technical Overview

This project showcases advanced digital design techniques, including:

  • Finite State Machine (FSM): A state-driven architecture for managing input capture, stack operations, and output generation.
  • Stack-Oriented Processing: Utilizes a hardware-based stack to store intermediate results and operators for sequential evaluation.
  • Arithmetic Evaluation: Supports basic arithmetic operations such as addition, subtraction, and multiplication, with provisions for operator precedence.
  • Error Handling: Implements mechanisms to detect invalid inputs and ensure robust operation.

Key Features

  • Hardware-Based Stack: A custom-designed stack module for efficient operand and operator management.
  • Operator Precedence: Ensures correct evaluation order for complex expressions.
  • Modular Design: Separates the FSM, stack, and arithmetic logic into distinct modules for scalability and maintainability.
  • Real-Time Processing: Processes inputs and generates outputs in real-time, demonstrating the speed of hardware-based computation.

Implementation Details

The IPN Calculator is implemented using Verilog and adheres to a modular design pattern:

  • FSM Module: Manages the overall control flow, including input capture, stack operations, and output generation.
  • Stack Module: A hardware-based stack for storing operands and operators, with support for push and pop operations.
  • Arithmetic Logic: Performs basic arithmetic operations and handles operator precedence.

The design is synthesized and tested on an FPGA, ensuring real-world applicability and performance.
